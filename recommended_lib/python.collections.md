## 1. `from collections import Counter`

**–ö–ª–∞—Å—Å `Counter` –≤ Python –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π —Å–ø–µ—Ü–∏–∞–ª–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Å–ª–æ–≤–∞—Ä—å, –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω–Ω—ã–π –¥–ª—è –ø–æ–¥—Å—á–µ—Ç–∞ —Ö–µ—à–∏—Ä—É–µ–º—ã—Ö –æ–±—ä–µ–∫—Ç–æ–≤**. –û–Ω –ø–æ–∑–≤–æ–ª—è–µ—Ç **–ª–µ–≥–∫–æ** –∏ **–±—ã—Å—Ç—Ä–æ** **–ø–æ–¥—Å—á–∏—Ç—ã–≤–∞—Ç—å –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –≤—Ö–æ–∂–¥–µ–Ω–∏–π —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—è—Ö**, —Ç–∞–∫–∏—Ö –∫–∞–∫ **—Å–ø–∏—Å–∫–∏**, **—Å—Ç—Ä–æ–∫–∏** –∏ **–∫–æ—Ä—Ç–µ–∂–∏**.

### –°–æ–∑–¥–∞–Ω–∏–µ –æ–±—ä–µ–∫—Ç–∞ `Counter`
- **–ü—Ä–∏ —Å–æ–∑–¥–∞–Ω–∏–∏ Counter –∏–∑ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏, –æ–Ω –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –ø–æ–¥—Å—á–∏—Ç—ã–≤–∞–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –≤—Ö–æ–∂–¥–µ–Ω–∏–π –∫–∞–∂–¥–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞**. –ï—Å–ª–∏ –ø–µ—Ä–µ–¥–∞—Ç—å **–ø—É—Å—Ç—É—é –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å**, –±—É–¥–µ—Ç —Å–æ–∑–¥–∞–Ω **–ø—É—Å—Ç–æ–π `Counter`**.
- –ï—Å–ª–∏ –ø–µ—Ä–µ–¥–∞—Ç—å –≤ `Counter` —Å–ª–æ–≤–∞—Ä—å, **–∫–ª—é—á–∏ –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å —Ö–µ—à–∏—Ä—É–µ–º—ã–º–∏**. –ï—Å–ª–∏ –ø–µ—Ä–µ–¥–∞—Ç—å **–Ω–µ —Ö–µ—à–∏—Ä—É–µ–º—ã–π –æ–±—ä–µ–∫—Ç, –±—É–¥–µ—Ç –≤—ã–∑–≤–∞–Ω–æ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ TypeError**.
- **–ü—Ä–∏–º–µ—Ä**:

#### 2. **–ú–µ—Ç–æ–¥ `most_common(n)`**:
- **–í–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –∏–∑ `n` –Ω–∞–∏–±–æ–ª–µ–µ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∏ –∏—Ö —Å—á–µ—Ç—á–∏–∫–æ–≤**. **–ï—Å–ª–∏ `n` –±–æ–ª—å—à–µ, —á–µ–º –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –≤–µ—Ä–Ω–µ—Ç –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã**.
- –ï—Å–ª–∏ **–ø–µ—Ä–µ–¥–∞—Ç—å –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –¥–ª—è n, –±—É–¥–µ—Ç –≤–æ–∑–≤—Ä–∞—â–µ–Ω –ø—É—Å—Ç–æ–π —Å–ø–∏—Å–æ–∫**. **–¢–∞–∫–∂–µ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å most_common() –±–µ–∑ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤, —á—Ç–æ–±—ã –ø–æ–ª—É—á–∏—Ç—å –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –≤ –ø–æ—Ä—è–¥–∫–µ —É–±—ã–≤–∞–Ω–∏—è —á–∞—Å—Ç–æ—Ç—ã**.

#### 3. **–ú–µ—Ç–æ–¥ `subtract()`**:
- **–ú–µ—Ç–æ–¥ `subtract()` –≤ –∫–ª–∞—Å—Å–µ `Counter` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —É–º–µ–Ω—å—à–µ–Ω–∏—è —Å—á–µ—Ç—á–∏–∫–æ–≤, –æ—Å–Ω–æ–≤—ã–≤–∞—è—Å—å –Ω–∞ –∑–Ω–∞—á–µ–Ω–∏—è—Ö –∏–∑ –¥—Ä—É–≥–æ–≥–æ Counter –∏–ª–∏ –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞**. –≠—Ç–æ—Ç –º–µ—Ç–æ–¥ –ø–æ–∑–≤–æ–ª—è–µ—Ç **–≤—ã—á–∏—Ç–∞—Ç—å –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –≤—Ö–æ–∂–¥–µ–Ω–∏–π —ç–ª–µ–º–µ–Ω—Ç–æ–≤, —á—Ç–æ –º–æ–∂–µ—Ç –±—ã—Ç—å –ø–æ–ª–µ–∑–Ω–æ –≤ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö —Å—Ü–µ–Ω–∞—Ä–∏—è—Ö**.
- **–ü—Ä–∏–º–µ—Ä –≤—ã—á–∏—Ç–∞–Ω–∏—è**: –£ –≤–∞—Å –µ—Å—Ç—å `Counter`, –∫–æ—Ç–æ—Ä—ã–π **–ø–æ–¥—Å—á–∏—Ç—ã–≤–∞–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Ñ—Ä—É–∫—Ç–æ–≤**, –∏ –≤—ã —Ö–æ—Ç–∏—Ç–µ **—É–º–µ–Ω—å—à–∏—Ç—å –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã—Ö —Ñ—Ä—É–∫—Ç–æ–≤**.

#### 4. **–ú–µ—Ç–æ–¥ `update()`**:
- **–û–±–Ω–æ–≤–ª—è–µ—Ç —Å—á–µ—Ç—á–∏–∫–∏, –¥–æ–±–∞–≤–ª—è—è –∑–Ω–∞—á–µ–Ω–∏—è –∏–∑ –¥—Ä—É–≥–æ–≥–æ Counter –∏–ª–∏ –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞**. –ú–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è **–¥–æ–±–∞–≤–ª–µ–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∏–∑ —Å–ø–∏—Å–∫–∞ –∏–ª–∏ –¥—Ä—É–≥–æ–≥–æ Counter**. –ï—Å–ª–∏ **—ç–ª–µ–º–µ–Ω—Ç –æ—Ç—Å—É—Ç—Å—Ç–≤—É–µ—Ç, –æ–Ω –±—É–¥–µ—Ç –¥–æ–±–∞–≤–ª–µ–Ω —Å –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ–º 1**. –ï—Å–ª–∏ –ø–µ—Ä–µ–¥–∞—Ç—å –∏—Ç–µ—Ä–∏—Ä—É–µ–º—ã–π –æ–±—ä–µ–∫—Ç, –∫–æ—Ç–æ—Ä—ã–π —Å–æ–¥–µ—Ä–∂–∏—Ç **–Ω–µ —Ö–µ—à–∏—Ä—É–µ–º—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã, –±—É–¥–µ—Ç –≤—ã–∑–≤–∞–Ω–æ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ TypeError**.

#### 5. **–ê—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏** 
- –ü–æ–¥–¥–µ—Ä–∂–∏–≤–∞—é—Ç—Å—è –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å–ª–æ–∂–µ–Ω–∏—è, –≤—ã—á–∏—Ç–∞–Ω–∏—è, –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏—è `(&)`, –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏—è `(|)`:
- **–ü—Ä–∏–º–µ—Ä**:
```py

print(counter1 & counter2)  # Counter({'a': 1, 'b': 1})  # –ú–∏–Ω–∏–º—É–º
print(counter1 | counter2)  # Counter({'b': 4, 'a': 3, 'c': 2})  # –ú–∞–∫—Å–∏–º—É–º
```
#### 5. –ò—Ç–æ–≥–æ–≤—ã–π –ø—Ä–∏–º–µ—Ä: –ê–Ω–∞–ª–∏–∑ —Ç–µ–∫—Å—Ç–∞

```py
# –¢–µ–∫—Å—Ç –¥–ª—è –∞–Ω–∞–ª–∏–∑–∞

In software development, counter tools are widely used for analytics. 
For example, counting the most frequently used words in a text is a common task.
"""

# –ü—Ä–µ–¥–æ–±—Ä–∞–±–æ—Ç–∫–∞ —Ç–µ–∫—Å—Ç–∞: –ø—Ä–∏–≤–µ–¥–µ–Ω–∏–µ –∫ –Ω–∏–∂–Ω–µ–º—É —Ä–µ–≥–∏—Å—Ç—Ä—É –∏ —Ä–∞–∑–±–∏–µ–Ω–∏–µ –Ω–∞ —Å–ª–æ–≤–∞

    print(f"{word}: {count}")

# –†–µ–∑—É–ª—å—Ç–∞—Ç:
# –¢–æ–ø-5 —Å–∞–º—ã—Ö —á–∞—Å—Ç—ã—Ö —Å–ª–æ–≤:

# software: 1
# development: 1
# counter: 1
# tools: 1
```

#### 1. `move_to_end(key, last=True)`
- **–ü–µ—Ä–µ–º–µ—â–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç —Å —É–∫–∞–∑–∞–Ω–Ω—ã–º –∫–ª—é—á–æ–º –ª–∏–±–æ –≤ –∫–æ–Ω–µ—Ü, –ª–∏–±–æ –≤ –Ω–∞—á–∞–ª–æ —Å–ª–æ–≤–∞—Ä—è**.

3. **–ö—ç—à–∏—Ä–æ–≤–∞–Ω–∏–µ (LRU-–∫—ç—à)**: –ö—ç—à –Ω–∞ –æ—Å–Ω–æ–≤–µ —Å—Ç—Ä–∞—Ç–µ–≥–∏–∏ "–Ω–∞–∏–º–µ–Ω–µ–µ –Ω–µ–¥–∞–≤–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–Ω—ã–π" (Least Recently Used).
```py
from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity: int):
        self.cache = OrderedDict()
        self.capacity = capacity

    def get(self, key: int) -> int:
        if key in self.cache:
            # –ü–µ—Ä–µ–º–µ—Å—Ç–∏—Ç—å —ç–ª–µ–º–µ–Ω—Ç –≤ –∫–æ–Ω–µ—Ü, —Ç–∞–∫ –∫–∞–∫ –æ–Ω –Ω–µ–¥–∞–≤–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª—Å—è
            self.cache.move_to_end(key)
            return self.cache[key]
        return -1

    def put(self, key: int, value: int) -> None:
        if key in self.cache:
            # –û–±–Ω–æ–≤–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –∏ –ø–µ—Ä–µ–º–µ—Å—Ç–∏—Ç—å –≤ –∫–æ–Ω–µ—Ü
            self.cache.move_to_end(key)
        else:
            if len(self.cache) >= self.capacity:
                # –£–¥–∞–ª–∏—Ç—å –Ω–∞–∏–º–µ–Ω–µ–µ –Ω–µ–¥–∞–≤–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç
                self.cache.popitem(last=False)
            self.cache[key] = value

# –ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è
lru = LRUCache(2)
lru.put(1, 1)
lru.put(2, 2)
print(lru.get(1))  # 1
lru.put(3, 3)  # –£–¥–∞–ª—è–µ—Ç –∫–ª—é—á 2
print(lru.get(2))  # -1 (–Ω–µ –Ω–∞–π–¥–µ–Ω)
lru.put(4, 4)  # –£–¥–∞–ª—è–µ—Ç –∫–ª—é—á 1
print(lru.get(1))  # -1 (–Ω–µ –Ω–∞–π–¥–µ–Ω)
print(lru.get(3))  # 3
print(lru.get(4))  # 4
```

### –ó–∞–¥–∞—á–∏ –¥–ª—è –∑–∞–∫—Ä–µ–ø–ª–µ–Ω–∏—è —Ç–µ–º—ã OrderedDict

## 4. `from collections import deque`

**`deque` (`double-ended queue`, –¥–≤—É—Å—Ç–æ—Ä–æ–Ω–Ω—è—è –æ—á–µ—Ä–µ–¥—å) ‚Äî —ç—Ç–æ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö, –æ–ø—Ç–∏–º–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–∞—è –¥–ª—è —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ–≥–æ –¥–æ–±–∞–≤–ª–µ–Ω–∏—è –∏ —É–¥–∞–ª–µ–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —Å –æ–±–æ–∏—Ö –∫–æ–Ω—Ü–æ–≤**.
- –û—Å–Ω–æ–≤–∞–Ω –Ω–∞ –¥–≤—É—Å–≤—è–∑–Ω–æ–º —Å–ø–∏—Å–∫–µ, —á—Ç–æ –¥–µ–ª–∞–µ—Ç –æ–ø–µ—Ä–∞—Ü–∏–∏ append –∏ pop —Å –æ–±–æ–∏—Ö –∫–æ–Ω—Ü–æ–≤ –æ—á–µ—Ä–µ–¥–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–Ω—ã–º–∏ –∑–∞ ùëÇ(1).
- –í –æ—Ç–ª–∏—á–∏–µ –æ—Ç —Å–ø–∏—Å–∫–∞ (list), –æ–ø–µ—Ä–∞—Ü–∏–∏ –≤—Å—Ç–∞–≤–∫–∏/—É–¥–∞–ª–µ–Ω–∏—è –≤ —Å–µ—Ä–µ–¥–∏–Ω–µ –æ—á–µ—Ä–µ–¥–∏ –º–µ–Ω–µ–µ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω—ã.

### –û—Å–Ω–æ–≤–Ω—ã–µ –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏
- **–≠—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ–µ –¥–æ–±–∞–≤–ª–µ–Ω–∏–µ –∏ —É–¥–∞–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —Å –æ–±–æ–∏—Ö –∫–æ–Ω—Ü–æ–≤**.
- **–í–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å —Ä–∞–±–æ—Ç—ã –∫–∞–∫ —Å –æ—á–µ—Ä–µ–¥—å—é, —Ç–∞–∫ –∏ —Å–æ —Å—Ç–µ–∫–æ–º**.
- **–û–ø—Ü–∏–æ–Ω–∞–ª—å–Ω–∞—è —É—Å—Ç–∞–Ω–æ–≤–∫–∞ –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–≥–æ —Ä–∞–∑–º–µ—Ä–∞ (`maxlen`), –ø–æ—Å–ª–µ –ø—Ä–µ–≤—ã—à–µ–Ω–∏—è –∫–æ—Ç–æ—Ä–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç—ã –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ —É–¥–∞–ª—è—é—Ç—Å—è —Å –ø—Ä–æ—Ç–∏–≤–æ–ø–æ–ª–æ–∂–Ω–æ–≥–æ –∫–æ–Ω—Ü–∞**.

### –ú–µ—Ç–æ–¥—ã `deque`
#### 1. –°–æ–∑–¥–∞–Ω–∏–µ deque
```python

# –ü—É—Å—Ç–∞—è –æ—á–µ—Ä–µ–¥—å

# –û—á–µ—Ä–µ–¥—å —Å –Ω–∞—á–∞–ª—å–Ω—ã–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏

dq = deque({1, 2, 3, 4, 5})
print(dq)  # deque([1, 2, 3, 4, 5])
dq = deque('apple')
print(dq)  # deque(['a', 'p', 'p', 'l', 'e'])

# –û—á–µ—Ä–µ–¥—å —Å –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏–µ–º —Ä–∞–∑–º–µ—Ä–∞

```

#### 2. –î–æ–±–∞–≤–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤
- `append(x)`: **–î–æ–±–∞–≤–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç x –≤ –∫–æ–Ω–µ—Ü –æ—á–µ—Ä–µ–¥–∏**.
- `appendleft(x)`: **–î–æ–±–∞–≤–ª—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç x –≤ –Ω–∞—á–∞–ª–æ –æ—á–µ—Ä–µ–¥–∏**.

#### 3. –£–¥–∞–ª–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤
- `pop()`: **–£–¥–∞–ª—è–µ—Ç –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø–æ—Å–ª–µ–¥–Ω–∏–π —ç–ª–µ–º–µ–Ω—Ç**.

#### 4. –ü—Ä–æ—Å–º–æ—Ç—Ä –∏ –∏–∑–º–µ–Ω–µ–Ω–∏–µ —ç–ª–µ–º–µ–Ω—Ç–æ–≤
- `clear()`: **–û—á–∏—â–∞–µ—Ç –æ—á–µ—Ä–µ–¥—å**.
- `index(x, start, stop)`: **–í–æ–∑–≤—Ä–∞—â–∞–µ—Ç –∏–Ω–¥–µ–∫—Å –ø–µ—Ä–≤–æ–≥–æ –ø–æ—è–≤–ª–µ–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç–∞ x**.

- `count(x)`: **–°—á–∏—Ç–∞–µ—Ç –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –≤—Ö–æ–∂–¥–µ–Ω–∏–π —ç–ª–µ–º–µ–Ω—Ç–∞ x**.

#### 5. –†–∞–±–æ—Ç–∞ —Å –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–Ω—ã–º —Ä–∞–∑–º–µ—Ä–æ–º
- `maxlen`: **–ï—Å–ª–∏ –∑–∞–¥–∞–Ω, –ø—Ä–∏ –¥–æ–±–∞–≤–ª–µ–Ω–∏–∏ –Ω–æ–≤–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ –ø—Ä–∏ –ø–æ–ª–Ω–æ–º –∑–∞–ø–æ–ª–Ω–µ–Ω–∏–∏ –æ—á–µ—Ä–µ–¥–∏ —É–¥–∞–ª—è–µ—Ç—Å—è —ç–ª–µ–º–µ–Ω—Ç —Å –ø—Ä–æ—Ç–∏–≤–æ–ø–æ–ª–æ–∂–Ω–æ–≥–æ –∫–æ–Ω—Ü–∞**.

#### 6. –ü–µ—Ä–µ—Å—Ç–∞–Ω–æ–≤–∫–∞ —ç–ª–µ–º–µ–Ω—Ç–æ–≤
- `extend(iterable)`: **–†–∞—Å—à–∏—Ä—è–µ—Ç –æ—á–µ—Ä–µ–¥—å —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏ –∏–∑ –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞, –¥–æ–±–∞–≤–ª—è—è –∏—Ö –≤ –∫–æ–Ω–µ—Ü**.
- `extendleft(iterable)`: **–†–∞—Å—à–∏—Ä—è–µ—Ç –æ—á–µ—Ä–µ–¥—å —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏ –∏–∑ –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞, –¥–æ–±–∞–≤–ª—è—è –∏—Ö –≤ –Ω–∞—á–∞–ª–æ (–≤ –æ–±—Ä–∞—Ç–Ω–æ–º –ø–æ—Ä—è–¥–∫–µ)**.
- `rotate(n)`: **–°–º–µ—â–∞–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã –≤–ø—Ä–∞–≤–æ –Ω–∞ n –ø–æ–∑–∏—Ü–∏–π (–ø—Ä–∏ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ–º n ‚Äî –≤–ª–µ–≤–æ)**.


### –û—Ç–ª–∏—á–∏—è –æ—Ç –æ–±—ã—á–Ω–æ–≥–æ —Å–ø–∏—Å–∫–∞
- **–î–æ–±–∞–≤–ª–µ–Ω–∏–µ –∏ —É–¥–∞–ª–µ–Ω–∏–µ —Å –æ–±–æ–∏—Ö –∫–æ–Ω—Ü–æ–≤**: –í —Å–ø–∏—Å–∫–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ append –∏ pop —Ä–∞–±–æ—Ç–∞—é—Ç —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ —Ç–æ–ª—å–∫–æ —Å –∫–æ–Ω—Ü–∞. –£ deque —ç—Ç–æ –≤–æ–∑–º–æ–∂–Ω–æ –∏ —Å –Ω–∞—á–∞–ª–∞.
- **–û–≥—Ä–∞–Ω–∏—á–µ–Ω–∏–µ –¥–ª–∏–Ω—ã (`maxlen`)**: –í —Å–ø–∏—Å–∫–µ —Ç–∞–∫–æ–π —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç–∏ –Ω–µ—Ç.
- **–û–ø—Ç–∏–º–∏–∑–∏—Ä–æ–≤–∞–Ω –¥–ª—è –æ—á–µ—Ä–µ–¥–µ–π**: –ü—Ä–∏ –¥–æ–±–∞–≤–ª–µ–Ω–∏–∏/—É–¥–∞–ª–µ–Ω–∏–∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å deque –≤—ã—à–µ, —á–µ–º —É —Å–ø–∏—Å–∫–∞.

```python
    Product('Apple', 'Fruits', 1.2, 50),
    Product('Orange', 'Fruits', 1.5, 30),
    Product('Carrot', 'Vegetables', 0.8, 100),
    Product('Potato', 'Vegetables', 0.5, 200),
    Product('Milk', 'Dairy', 2.0, 20),
    Product('Cheese', 'Dairy', 5.0, 10)

for p in products_dicts:
    print(p)
```

## 6. `from collections import ChainMap`

**–ú–æ–¥—É–ª—å `ChainMap` –∏–∑ –±–∏–±–ª–∏–æ—Ç–µ–∫–∏ `collections` –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–π –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä –¥–ª—è –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏—è –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö —Å–ª–æ–≤–∞—Ä–µ–π –∏–ª–∏ –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–π –≤ –æ–¥–Ω—É –ª–æ–≥–∏—á–µ—Å–∫—É—é —Å—Ç—Ä—É–∫—Ç—É—Ä—É**. –û–Ω –ø–æ–∑–≤–æ–ª—è–µ—Ç **—Ä–∞–±–æ—Ç–∞—Ç—å —Å –æ–±—ä–µ–¥–∏–Ω–µ–Ω–Ω—ã–º–∏ –¥–∞–Ω–Ω—ã–º–∏ –∫–∞–∫ —Å –æ–¥–Ω–∏–º –æ–±—ä–µ–∫—Ç–æ–º, —Å–æ—Ö—Ä–∞–Ω—è—è –ø—Ä–∏ —ç—Ç–æ–º –¥–æ—Å—Ç—É–ø –∫ –∫–∞–∂–¥–æ–º—É –∏–∑ –æ—Ä–∏–≥–∏–Ω–∞–ª—å–Ω—ã—Ö —Å–ª–æ–≤–∞—Ä–µ–π**.

### –ö–ª—é—á–µ–≤—ã–µ –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏ `ChainMap`:

- **–ü–æ—Ä—è–¥–æ–∫ –ø–æ–∏—Å–∫–∞**: **–ü—Ä–∏ –æ–±—Ä–∞—â–µ–Ω–∏–∏ –∫ –∫–ª—é—á—É –ø–æ–∏—Å–∫ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –≤ –ø–æ—Ä—è–¥–∫–µ –¥–æ–±–∞–≤–ª–µ–Ω–∏—è —Å–ª–æ–≤–∞—Ä–µ–π. –ü–µ—Ä–≤—ã–π —Å–ª–æ–≤–∞—Ä—å –∏–º–µ–µ—Ç –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç**.

### –°–æ–∑–¥–∞–Ω–∏–µ –æ–±—ä–µ–∫—Ç–∞ ChainMap
