if __name__ == "__main__":

## 7. JOIN-–∑–∞–ø—Ä–æ—Å—ã: —Ç–µ–æ—Ä–µ—Ç–∏—á–µ—Å–∫–∏ –∏ –Ω–∞ Python

### –ß—Ç–æ —Ç–∞–∫–æ–µ `JOIN`?

`JOIN` ‚Äî —ç—Ç–æ —Å–ø–æ—Å–æ–± –æ–±—ä–µ–¥–∏–Ω–∏—Ç—å –¥–∞–Ω–Ω—ã–µ –∏–∑ –¥–≤—É—Ö (–∏–ª–∏ –±–æ–ª–µ–µ) —Ç–∞–±–ª–∏—Ü, –æ—Å–Ω–æ–≤—ã–≤–∞—è—Å—å –Ω–∞ –ª–æ–≥–∏—á–µ—Å–∫–æ–π —Å–≤—è–∑–∏ –º–µ–∂–¥—É –Ω–∏–º–∏. –ß–∞—â–µ –≤—Å–µ–≥–æ ‚Äî –ø–æ FOREIGN KEY.

### –ß—Ç–æ –¥–µ–ª–∞–µ—Ç `JOIN`?

–≠—Ç–æ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ (`INNER JOIN`). –û–Ω–æ –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç —Ç–æ–ª—å–∫–æ —Ç–µ –∑–∞–ø–∏—Å–∏, –≥–¥–µ –Ω–∞–π–¥–µ–Ω–∞ –ø–∞—Ä–∞ –≤ –æ–±–µ–∏—Ö —Ç–∞–±–ª–∏—Ü–∞—Ö.

### **–ü—Ä–∏–º–µ—Ä 1: JOIN (INNER JOIN –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é): –Ω–∞–π—Ç–∏ –≤—Å–µ –∑–∞–∫–∞–∑—ã –∏ –∏–º–µ–Ω–∞ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π**

```python
cursor.execute("""
SELECT users.name, orders.item
FROM orders

""")

for row in cursor.fetchall():
    print(row)
```

### üìò –ü–æ—à–∞–≥–æ–≤–æ–µ –æ–±—ä—è—Å–Ω–µ–Ω–∏–µ:

1. `FROM orders` ‚Äî –æ—Å–Ω–æ–≤–Ω–∞—è —Ç–∞–±–ª–∏—Ü–∞: –±–µ—Ä—ë–º –≤—Å–µ —Å—Ç—Ä–æ–∫–∏ –∏–∑ orders.

2. `JOIN users` ‚Äî –ø—Ä–∏—Å–æ–µ–¥–∏–Ω—è–µ–º —Ç–∞–±–ª–∏—Ü—É users.

3. `ON users.id = orders.user_id` ‚Äî —É—Å–ª–æ–≤–∏–µ —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è: –±–µ—Ä—ë–º —Ç–æ–ª—å–∫–æ —Ç–µ —Å—Ç—Ä–æ–∫–∏, –≥–¥–µ `orders.user_id` —Å–æ–≤–ø–∞–¥–∞–µ—Ç —Å `users.id`.

4. `SELECT users.name, orders.item` ‚Äî –≤—ã–±–∏—Ä–∞–µ–º –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–µ –ø–æ–ª—è: –∏–º—è –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è –∏ –∑–∞–∫–∞–∑.

5. `fetchall()` ‚Äî –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å–ø–∏—Å–æ–∫ –∫–æ—Ä—Ç–µ–∂–µ–π —Å —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞–º–∏ –∑–∞–ø—Ä–æ—Å–∞.

### **–ü—Ä–∏–º–µ—Ä 2: LEFT JOIN (–∏–ª–∏ LEFT OUTER JOIN): –ø–æ–∫–∞–∑–∞—Ç—å –≤—Å–µ—Ö –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π –¥–∞–∂–µ –±–µ–∑ –∑–∞–∫–∞–∑–æ–≤**

### üìò –ü–æ—à–∞–≥–æ–≤–æ–µ –æ–±—ä—è—Å–Ω–µ–Ω–∏–µ:

1. `FROM users` ‚Äî –æ—Å–Ω–æ–≤–Ω–∞—è —Ç–∞–±–ª–∏—Ü–∞ —Ç–µ–ø–µ—Ä—å users.

2. `LEFT JOIN orders` ‚Äî –ø—Ä–∏—Å–æ–µ–¥–∏–Ω—è–µ–º —Ç–∞–±–ª–∏—Ü—É orders, –≤–∫–ª—é—á–∞—è –≤—Å–µ—Ö –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π, –¥–∞–∂–µ –µ—Å–ª–∏ –∑–∞–∫–∞–∑–æ–≤ —É –Ω–∏—Ö –Ω–µ—Ç.

3. `ON users.id = orders.user_id` ‚Äî —Å–æ–µ–¥–∏–Ω—è–µ–º –ø–æ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏—é ID.

4. `SELECT users.name, orders.item` ‚Äî –≤—ã–≤–æ–¥–∏–º –∏–º—è –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è –∏ –∑–∞–∫–∞–∑ (–µ—Å–ª–∏ –µ—Å—Ç—å).

#### –ß—Ç–æ –¥–µ–ª–∞–µ—Ç LEFT JOIN?

`LEFT JOIN` –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç –≤—Å–µ—Ö –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π, –¥–∞–∂–µ –µ—Å–ª–∏ —É –Ω–∏—Ö –Ω–µ—Ç –∑–∞–∫–∞–∑–æ–≤. –î–ª—è —Ç–∞–∫–∏—Ö –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª–µ–π –∑–Ω–∞—á–µ–Ω–∏—è –∏–∑ orders –±—É–¥—É—Ç `NULL`.

### –°—Ä–∞–≤–Ω–µ–Ω–∏–µ —Ç–∏–ø–æ–≤ JOIN

- `INNER JOIN`: –¢–æ–ª—å–∫–æ —Å–æ–≤–ø–∞–¥–∞—é—â–∏–µ –∑–∞–ø–∏—Å–∏ (–æ–±–∞ –ø–æ–ª—è –Ω–µ NULL)
- `LEFT JOIN`: –í—Å–µ –∏–∑ –ª–µ–≤–æ–π —Ç–∞–±–ª–∏—Ü—ã, + —Å–æ–≤–ø–∞–¥–∞—é—â–∏–µ –∏–∑ –ø—Ä–∞–≤–æ–π, –µ—Å–ª–∏ –µ—Å—Ç—å
- `RIGHT JOIN`: –í—Å–µ –∏–∑ –ø—Ä–∞–≤–æ–π —Ç–∞–±–ª–∏—Ü—ã, + —Å–æ–≤–ø–∞–¥–∞—é—â–∏–µ –∏–∑ –ª–µ–≤–æ–π (–≤ SQLite –Ω–µ—Ç, —ç–º—É–ª–∏—Ä—É–µ—Ç—Å—è)
- `FULL JOIN`: –í—Å–µ –∑–∞–ø–∏—Å–∏ –∏–∑ –æ–±–µ–∏—Ö —Ç–∞–±–ª–∏—Ü (SQLite –Ω–µ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –Ω–∞–ø—Ä—è–º—É—é, –Ω—É–∂–µ–Ω UNION)

## üí° –î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–∞—è –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è
